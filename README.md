# PRODIGY_GA_01 – Text Generation with GPT-2

## Project Overview

This project was developed as part of the **Generative AI Internship Program at Prodigy InfoTech**. The objective is to implement a text generation system using the **GPT-2 (Generative Pre-trained Transformer 2)** model to generate coherent and contextually relevant text based on user-provided prompts.

The project demonstrates the practical application of transformer-based language models in **Natural Language Processing (NLP)** and Generative AI.

---

## Objective

To explore and implement AI-powered text generation using a pre-trained GPT-2 model and understand the fundamentals of transformer architectures and language modeling.

---

## Technologies and Tools

* Python
* Google Colab
* Hugging Face Transformers
* GPT-2
* PyTorch

---

## Project Features

* Generates human-like text from custom prompts
* Utilizes a pre-trained GPT-2 language model
* Demonstrates transformer-based language generation
* Provides efficient and scalable text generation capabilities

---

## Implementation Process

1. Installed the required Python libraries.
2. Loaded the pre-trained GPT-2 model using the Hugging Face Transformers library.
3. Provided custom text prompts as input.
4. Generated contextually relevant text continuations.
5. Saved and documented the generated outputs.

---

## Sample Input

```text
Artificial Intelligence is
```

## Sample Output

